# Instructions


## Install Dependecies

    yarn

## Setup database in "ormconfig.json"

```
    "host": "localhost",
    "port": "5433",
    "username": "postgres",
    "password": "docker",
    "database": "graphqlchallenge",
```

##  Run migrations

    yarn typeorm migrations:run

## Start App in development mode

    yarn dev:server
